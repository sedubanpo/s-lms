학생 변경 요청

Firestore students 문서에 직접 저장할 payload를 만듭니다.

생년 기반 학년 자동 계산생년을 입력하면 1월 1일 기준 학년을 계산해 학년 입력값에 반영합니다.

입력 후 대기열에 추가하고, 관리자 세션이면 해당 요청만 즉시 Firestore에 반영합니다. 실패한 요청은 대기열에 남아 재시도할 수 있습니다.

강사 계정 요청

Firestore 계정 문서와 로그인 alias를 갱신합니다.

Firebase Auth현재 비밀번호는 조회할 수 없습니다. 신규 계정은 입력한 비밀번호로 생성되며, 기존 계정은 관리자 비밀번호 재설정 절차를 사용합니다.

신규 강사는 Auth, custom claim, Firestore, Supabase 권한 연결이 모두 완료된 뒤에만 생성 완료로 표시합니다.
