학생 데이터

담임 학생 대시보드

수업 권한과 별도로, 선생님별로 실제 담당하는 학생을 빠르게 찾아 다중 선택으로 배정하고 한눈에 관리합니다.

선택 학생 0명

권한 관리

학생명을 검색하고, 원/학년으로 좁힌 뒤 학생을 선택하면 우측에서 수업 열람 권한을 빠르게 관리할 수 있습니다.

직보 매니저

학교별 시험 전날 직보 배정과 과목 사전, 학생별 예외를 한 화면에서 관리합니다.

직보 보드

주요일정의 시험 이벤트를 읽어 학교/학년별 직보 배정을 월 기준으로 정리합니다.

과목 사전

시험 과목명과 학생과목키워드로 시험 이벤트를 매칭하고, 강사과목분류를 함께 관리합니다.

등록된 시험 과목명과 학생과목키워드는 직보 보드 자동 매칭에 바로 반영됩니다.
직보 과목 수정 모드

선택한 직보 과목 정보를 수정한 뒤 저장하세요.

학생별 보완 / 오버라이드

학생별 과목 포함/제외, 전용 강사 지정, 직보일자 수동 오버라이드를 저장합니다.

학생별 직보 예외 수정 모드

선택한 학생별 직보 예외를 수정한 뒤 저장하세요.

미배정 직보 강사 지정

현재 월 보드에서 강사가 자동 배정되지 않은 직보에 전용 강사를 바로 지정합니다.

학교와 학년을 대상에 함께 입력하면 캘린더에서 가장 먼저 표시됩니다.

자료 수정 모드

선택한 자료 정보를 수정한 뒤 저장하세요.

강사 학생 열람 권한 설정

활성 학생을 선택한 뒤, 해당 학생을 열람할 수 있는 강사를 체크해 저장하세요. 저장 즉시 강사 계정 화면에 반영됩니다.

강사 계정에서 학생이 0명으로 표시되면 아직 권한이 연결되지 않은 상태일 수 있습니다. 이 설정창에서 학생과 강사를 연결해 주세요.

상담/관리 기록 상세보기

선택한 학생 기록의 전체 내용과 작성 정보를 확인합니다.

학생 상세 보기

상담, 권한, 시험, 내신 자료 연결 상태를 한 화면에서 확인합니다.

문제 자료실 설정

시험구분, 자료유형, 파일유형 옵션을 직접 추가할 수 있습니다.

추가한 옵션은 자료 등록 선택창에 바로 반영됩니다.
시험구분 예: 월말평가, 반배치고사
자료유형 예: 개념정리, 서술형특강
파일유형 예: 해설지, 첨삭본

최근 자료 열람 기록

강사별 문제 자료실 열람 이력을 최근 순으로 확인합니다.

직보 진행 상태

선택한 직보의 상태, 시간, 특이사항을 저장합니다.

직보 정보를 불러오는 중입니다.

미확정 직보 매칭
